The `namefully` package is available on [npm](https://www.npmjs.com/package/namefully)
and can be installed using the following commands:

<Tabs
defaultValue="npm"
values={[
{ label: 'npm', value: 'npm', },
{ label: 'Yarn', value: 'yarn', },
]
}>
<TabItem
    value="npm">

```sh
npm i namefully@1.0.6
```

</TabItem>
<TabItem value="yarn"
>

```sh
yarn add namefully@1.0.6
```

</TabItem>

</Tabs>

## Dependencies

None
